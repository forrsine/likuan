## 今日完成

- 新增 `regex_compile_ex`，编译失败时可直接获得详细错误文本。
- 新增 `regex_status_string`，把错误码转换为可读说明。
- `rx_cli` 现在同时输出错误位置、错误码和错误类别。
- 新增 `rx_dump_tokens`，显示 token 的字节位置、类型和值。
- 增加 256 层括号嵌套限制，防止递归解析耗尽调用栈。
- 增加括号、字符类、区间量词、重复运算和未知 flags 的精确错误码测试。
