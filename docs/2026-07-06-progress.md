## 设计说明

字符是否等价，不按字符编码是否相邻判断，而按它们经过所有 NFA 字符转移时的行为是否完全一致判断。例如：

```text
pattern                    character classes
a*                         2: a / other
a|b                        3: a / b / other
([a-z]+)\d{2,4}           3: letter / digit / other
```

对于锚点，DFA 构造不把 `^`、`$` 当作普通字符，而是在 closure 中依据 `at_begin`、`at_end` 条件决定是否经过对应零宽边。这样无需在匹配时退回 NFA。
